### 2. Sequece(자료형)

> list, tuple

- list

  - 임의의 데이터를 순서대로 저장하는 집합 자료형
  - 대괄호([])나  list()를 이용하여 표시

```python
# 리스트 생성
>>> lst = list()
>>> print(lst)
[]
  
>>> lst = []
>>> print(lst)
[]
  
>>> lst = [1, 2, 3, 4.5, True]
>>> print(lst)
[1, 2, 3, 4.5, True]
  
>>> lst = [1, 2, ['Hi', ":)"], 28.0, False]
>>> print(lst)
[1, 2, ['Hi', ":)"], 28.0, False]

# 리스트 슬라이싱(slicing)

>>> print(lst[0])
1
>>> print(lst[-1])
False
>>> print(lst[::2])
[1, ['Hi', ':)'], False]
```

- tuple
  - list와 상당히 유사하나 읽기만 가능
  - 소괄호(()), tuple()을 이용해서 표현

```python
>>> tup = tuple()
>>> print(tup)
()

>>> tup = ()
>>> print(tup)
()

>>> lst = [1] # 요소 1개만 가지고 있는 list
>>> tup = [1,] # 요소 1개만 가지고 있는 tuple

# tuple은 소괄호를 생략할 수 있다.
>>> tup = 1, 2, 3
>>> print(type(tup))
<class 'tuple'>

# list to tuple, tuple to list
>>> lst = 4, 5, 6
>>> ret = tuple(lst)
>>> print(ret)
(4, 5, 6)

>>> ret = list(ret)
>>> print(ret)
[4, 5, 6]
```
